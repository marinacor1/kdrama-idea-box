An idea box focused on organizing KDrama favorites.

To use:

Git clone application.

Run bundle

Run rake db:create

Run rake db:migrate

Run rails s

Visit http://localhost:3000

Steps to use application:

1. Sign up for free/ Login if you already have an account

2. Click on "New Idea" for a new idea

3. Click on "All Ideas" to view all ideas

4. Edit your account.

5. If you would like to cancel, delete account.
